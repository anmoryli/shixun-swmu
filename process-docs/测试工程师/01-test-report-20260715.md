## 6. 已知限制与风险

- 当前主机无 Docker 命令，且未提供数据库、Redis、`BASE_URL`、管理员用户名和密码；因此未启动完整部署，也未执行会创建/清理业务数据的实时 57 用例 API 回归。历史证据位于 `process-docs/evidence/api/latest-summary.md`，时间为 2026-07-13，仅作前次基线。

- 后端编译存在 `AmapService` 未检查泛型转换提示；当前测试覆盖其成功与失败分支，建议后续用明确响应 DTO 消除警告。
- 前端页面组件、真实浏览器路由和地图交互尚无组件/E2E 自动化；本轮以接口契约、状态逻辑与生产构建兜底，不能替代浏览器兼容性和视觉验收。
