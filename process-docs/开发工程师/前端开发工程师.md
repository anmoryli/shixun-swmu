# 前端开发工程师

> 本文档由前端开发工程师视角，基于 Git 提交历史重新撰写，记录慧医数字医疗应用系统前端的开发历程、技术决策与交付成果。

## 一、角色职责

负责管理系统前端的架构迁移、页面开发、登录态改造、数据可视化与地图集成，保证前端与后端 API 契约一致、构建可部署、交互符合需求规格说明书。

## 四、关键工作与成果

1. **Vue 3 + Vite 迁移**：从 Vue 2 全量迁移到 Vue 3.5 + Vite 6，修复 require/ESM、CORS、SPA 404、TDZ 等迁移期问题，生产构建通过。
2. **持久登录且前端零 Token**：配合后端 httpOnly cookie，前端不再持有/存储任何 token；登录态由内存标志 + `/api/session` 探测恢复，路由守卫异步化。
3. **首页数据面板**：Echarts 渲染医生职称柱状图、治疗类型饼图，消费 `/api/dashboard` 的 counts/doctorLevels/treatTypes/news，含 resize 自适应与组件卸载释放。
4. **高德地图集成**：SaleManage 地图展示与地图选址新增/修改，独立 SaleMap 菜单页，Vuex 独立 state 避免覆盖分页。
5. **同源部署**：nginx:alpine 同源发布 + 反代 /api，消除跨域与 localhost 硬编码。
6. **代码规范**：闭环 CodeArts 静态扫描全部 G.* 系列前端 finding。
