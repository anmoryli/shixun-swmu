# medical-managerment-system

基于 Vue 3、Vite、Vue Router 4、Vuex 4 和 Element Plus 的管理端前端。

## Project setup
```
npm ci
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Preview production build
```
npm run preview
```
